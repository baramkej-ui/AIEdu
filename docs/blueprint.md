# **App Name**: AIEdu Teacher

## Core Features:

- 로그인 인증: Firebase 인증을 사용하여 이메일과 비밀번호로 교사를 인증하고, 역할 기반 접근 제어를 제공합니다.
- 학생 목록 관리: 학생 목록을 관리하고, Firestore에 저장된 학생들의 진행 상황 및 세부 정보를 확인합니다.
- Firebase 통합: 데이터 저장을 위해 Firebase Firestore를 사용하고, 사용자 관리를 위해 Firebase 인증을 사용하며, 배포를 위해 Firebase Hosting을 사용합니다.

## Style Guidelines:

- 기본 색상: 신뢰와 지성을 나타내는 밝은 파란색 (#ADD8E6)을 사용합니다.
- 배경 색상: 깨끗하고 읽기 쉬운 인터페이스를 위해 흰색 (#F9F8F7)을 사용합니다.
- 강조 색상: 성공 및 긍정적인 피드백을 나타내는 부드러운 녹색 (#90EE90)을 사용합니다.
- 본문 및 제목 글꼴: 깨끗하고 현대적인 읽기 환경을 위해 'Inter' sans-serif를 사용합니다.
- 코드 글꼴: 코드 스니펫 표시에 'Source Code Pro'를 사용합니다.
- FontAwesome 또는 Material Icons와 같은 세트에서 간단하고 전문적인 아이콘을 사용합니다.
- 일관된 간격과 정렬을 사용하여 깨끗하고 체계적인 레이아웃을 유지합니다.
